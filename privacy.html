Privacy Policy

Last updated: March 27, 2026

🛡️ Peak Power Table does not collect, store, or transmit any personal data. All data processing happens entirely on your Garmin device.

1. Overview

Peak Power Table ("the app") is a Garmin Connect IQ data field developed by Matthew Guenter, an independent developer operating under the name Peak Power Table. This privacy policy describes how the app handles data on your device.

2. Data Accessed

In order to display real-time performance metrics during an activity, the app reads the following data directly from your Garmin device:

  • Current power output — sourced from a connected power meter via ANT+/BLE during an active session
  • Current heart rate — sourced from a connected heart rate sensor or optical sensor during an active session
  • User body weight — read from your Garmin user profile to calculate watts per kilogram (W/kg). If unavailable, a manually configured value from the app settings is used instead.
  • User FTP setting — a manually entered value stored in the app's settings on your device
  • Interval configuration — a user-defined list of interval durations stored in the app's settings on your device

3. How Data Is Used

All data listed above is used solely to calculate and display peak power metrics on your Garmin device's screen during an activity. Specifically:

  • Power data is used to compute rolling peak average watts for each configured interval
  • Heart rate data is used to display average HR during your best power window for each interval
  • Body weight is used to calculate and display W/kg values
  • FTP and interval settings are used to configure the display

4. Data Storage and Transmission

The app does not store activity data beyond the current session. When your activity ends, all in-memory calculations are cleared. The app does not:

  • Transmit any data to external servers or third parties
  • Store activity data in persistent files on your device
  • Connect to the internet or any remote service
  • Share any data with the developer or any other party
  • Access data from previous activities

5. Garmin Connect IQ Platform

Peak Power Table is distributed through the Garmin Connect IQ Store and operates within the Garmin Connect IQ platform. Your use of the Garmin platform is subject to Garmin's Privacy Policy. This developer policy covers only the data handling within the app itself.

6. Children's Privacy

The app does not knowingly collect any data from any user, including minors. As no data is collected or transmitted, there are no special provisions required for children's privacy.

7. Changes to This Policy

If this privacy policy is updated in the future, the revised version will be posted at this URL with an updated "Last updated" date. Continued use of the app after any changes constitutes acceptance of the updated policy.
